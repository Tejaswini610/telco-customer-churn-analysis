# 📊 Telco Customer Churn Analysis

## 📌 Project Overview
This project focuses on analyzing customer churn patterns in a telecom company using data analytics techniques.  
The dashboard provides insights into customer behavior, subscription trends, contract types, payment methods, and factors influencing churn.

The main objective is to identify key reasons behind customer churn and help businesses improve customer retention strategies.

---

## 📂 Dataset Information
The dataset contains customer-related information such as:
- Customer demographics
- Internet services
- Contract details
- Payment methods
- Monthly & total charges
- Churn status

---

## 🛠 Tools Used
- Microsoft Excel / Libreoffice
- Pivot Tables
- Charts & Graphs
- Data Cleaning
- Dashboard Design

---

## 📊 Dashboard Features
- Overall Churn Analysis
- Customer Demographics Overview
- Contract Type Analysis
- Monthly Charges Insights
- Payment Method Comparison
- Internet Service Analysis
- Interactive Visualizations

---

## 🔍 Key Insights
- Customers with month-to-month contracts have higher churn rates.
- Fiber optic internet users showed increased churn.
- Electronic check payment method had more churned customers.
- Long-term contracts improved customer retention.
- Higher monthly charges contributed to churn in some cases.
